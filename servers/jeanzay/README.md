# Connecting to JeanZay


1. Add sshuttle - to create vpn, using the config which have the ip addresses of `jeanzay`


```bash
sshuttle -
```

2. Allocate a node on jeanzay

* allocate node
* get node
* get ip address
* disable checking
* start conda
